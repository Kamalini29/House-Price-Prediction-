# House-Price-Prediction-ML project

Complete end-to-end regression project (R²: 0.88+)

##  Results
| Model | R² Score | RMSE |
|-------|----------|------|
| XGBoost | 0.88 | 0.13 |
| Random Forest | 0.86 | 0.14 |

##  Tech Stack
- Pandas, NumPy, Scikit-Learn
- XGBoost, Matplotlib, Seaborn
- Kaggle Ames Housing Dataset

##  Quick Start
1. Clone repo
2. Upload `train.csv` 
3. `pip install -r requirements.txt`
4. Run `House_Price_Prediction.ipynb`
